0. <!DOCTYPE html>
    1) 문서의 HTML 버전을 지정
        -Document Type Definition(DTD)
        -HTML5(표준)
        -웹브라우저가 어떤 HTML 버전의 해석 방식으로 페이지를 이해하면 되는지를 알려주는 용도

1. <html></html>
    1) 문서의 전체 범위
    2) HTML 문서가 어디에서 시작하고, 어디에서 끝나는지 알려주는 역할

2. <head></head>
    1) 웹브라우저가 해석해야 할 정보(웹 페이지의 제목, 설명, 사용할 파일 위치, 
       스타일(CSS)같은 웹페이지의 보이지 않는)를 작성하는 범위
    2) <meta />
        -HTML 문서(웹페이지)의 제작자, 내용, 키워드같은 여러 정보를 검색엔진이나 브라우저에게 제공
    3) <title></title>
        -HTML 문서의 제목을 정의
        -웹브라우저의 탭에 표시됨
    4) <link />
        -외부 문서를 가져와 연결할 때 사용
        -대부분 CSS파일
        -<link rel="stylesheet" href="./main.css" />
            -rel: 가져 올 문서와 관계
            -href: 가져 올 문서의 경로
    5) <style></style>
        -스타일(CSS)을 HTML 문서 안에서 작성하는 경우에 사용 
    6) <script></script>
        -자바스크립트(JS) 파일을 가져오는 경우
        -자바스크립트(JS)를 HTML문서 안에서 작성하는 경우 

3. <body></body>
    1) 사용자 화면을 통해 보여지는 로고, 헤더, 푸터, 네비게이션, 메뉴, 버튼, 이미지같은 
       웹페이지의 보여지는 구조를 작성하는 범위
    2) <div></div>
        -블록(상자) 요소 
        -특별한 의미가 없는 구분을 위한 요소(Division)
       <h1></h1>
        -제목을 의미하는 요소(Heading)
       <p></p>
        -블록(상자) 요소
        -문장을 의미하는 요소(Paragraph)
       <img src='' alt='' />
        -인라인(글자) 요소
        -이미지를 삽입하는 요소(Image)
       <ul> <== 블록(상자) 요소, 순서가 필요없는 목록의 집합을 의미(Unordered List)
        <li>사과</li>  <== 블록(상자) 요소, 목록 내 각 항목(List Item)
        <li>딸기</li>
        <li>수박</li>
        <li>오렌지</li>
       </ul>
       <a></a>
        -인라인(글자) 요소 
        -다른/같은 페이지로 이동하는 하이퍼링크를 지정하는 요소 
        -Anchor(닻) 
       <span></span.
        -인라인(글자) 요소
        -특별한 의미가 없는 구분을 위한 요소
